# AI_2048
AI playable CLI based 2048 game.

Requirements: pynput module
Install by this command:
    $ pip install pynput

Run game:
    $ python main.py <mode> <ai_level>   
#mode 0 : AI plays, mode 1: Manual Gameplay, ai_level in [1,10)

**Probability of 2048 score**
    On 2nd level AI: 20% +
    On 4th level AI: 50% +
